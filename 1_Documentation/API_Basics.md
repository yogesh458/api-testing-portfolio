
# API Basics

API (Application Programming Interface) allows communication between client and server systems.

## Components of API
- URL (Endpoint): Address where API is hosted
- Method: GET, POST, PUT, DELETE
- Headers: Metadata like Content-Type, Authorization
- Request Body: Data sent to server
- Response: Data returned from server

## Example
POST /login
{
  "username": "user1",
  "password": "Pass@123"
}
